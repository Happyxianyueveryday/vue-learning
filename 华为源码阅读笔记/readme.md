  在由多个组件组成的Vue应用中，通常声明的组件需要进行导出，一般使用的定义组件的格式为:
  
  ```
  <script>
export default {
    name: 'id', // 组件名
    props: {    // 组件的子组件
        ...
    },
    data: {     // 组件属性
        ...
    },
    methods: {  // 组件方法
        ...
    }
    ...
}
</script>
  ```
